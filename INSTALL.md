# Установка плагина "Туристическая карта Кольчугино"

## Быстрая установка

### 1. Копирование файлов

```bash
# Скопируйте папку wordpress-plugin в WordPress
cp -r wordpress-plugin /path/to/your/wordpress/wp-content/plugins/kolchugino-map
```

Или через FTP/SFTP:
- Загрузите папку `wordpress-plugin` в `/wp-content/plugins/`
- Переименуйте в `kolchugino-map`

### 2. Активация

1. Войдите в админку WordPress: `yoursite.com/wp-admin`
2. **Плагины** → Найдите **"Кольчугино — Туристическая карта"**
3. Нажмите **Активировать**

### 3. Проверка установки

После активации в меню должен появиться пункт **"Туристическая карта"**

---

## Установка через WP-CLI

```bash
# Перейдите в директорию WordPress
cd /path/to/your/wordpress

# Скопируйте плагин
cp -r /path/to/kolchugino-map/wordpress-plugin wp-content/plugins/kolchugino-map

# Активируйте
wp plugin activate kolchugino-map
```

## Начальная настройка

### 1. Добавление первого объекта

1. **Туристическая карта** → **Добавить новый объект**
2. Заполните:
   - **Название:** "Пример кафе"
   - **Описание:** "Описание кафе"
   - **Категория:** Кафе и рестораны
   - **Координаты:** Кликните на карту или введите вручную
   - **Миниатюра:** Загрузите фото
   - **Контакты:** Телефон, сайт
3. Нажмите **Опубликовать**

### 2. Размещение карты на странице

1. Создайте новую страницу: **Страницы** → **Добавить новую**
2. Название: "Туристическая карта"
3. В контент вставьте шорткод:
   ```
   [kolchugino_map]
   ```
4. Нажмите **Опубликовать**

### 3. Проверка фронтенда

Откройте созданную страницу и убедитесь, что:
- Карта отображается
- Маркеры видны
- Фильтры работают
- Поиск функционирует
- Попапы открываются

## Требования к серверу

- **WordPress:** 5.0 или выше
- **PHP:** 7.4 или выше
- **MySQL:** 5.6 или выше / **MariaDB:** 10.1 или выше
- **Память PHP:** минимум 128MB

## Совместимость с темами

Плагин совместим с большинством тем WordPress. Если возникают проблемы с отображением:

1. Проверьте, что тема подключает jQuery
2. Убедитесь, что нет конфликтов CSS
3. При необходимости добавьте в `functions.php` темы:

```php
// Принудительная загрузка стилей плагина
add_action('wp_enqueue_scripts', function() {
    wp_enqueue_style('kolchugino-map-css');
    wp_enqueue_script('kolchugino-map-js');
}, 20);
```

## Удаление

1. **Плагины** → Деактивируйте плагин
2. Удалите плагин
3. При необходимости удалите данные из БД:

```sql
-- Удаление постов плагина
DELETE p, pm FROM wp_posts p
LEFT JOIN wp_postmeta pm ON p.ID = pm.post_id
WHERE p.post_type = 'map_object';

-- Удаление таксономии
DELETE t, tt, tr FROM wp_terms t
INNER JOIN wp_term_taxonomy tt ON t.term_id = tt.term_id
LEFT JOIN wp_term_relationships tr ON tt.term_taxonomy_id = tr.term_taxonomy_id
WHERE tt.taxonomy = 'map_category';
```

**Внимание:** Это удалит все данные плагина!
