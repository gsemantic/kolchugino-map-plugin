# Кольчугино — Туристическая карта v1.0.45

Плагин для WordPress, создающий интерактивную туристическую карту Кольчугинского района с достопримечательностями, кафе, магазинами и гостиницами.

## Установка

### 1. Загрузка плагина

1. Скачайте ZIP-архив `kolchugino-map-plugin.zip`
2. В админке WordPress перейдите в раздел "Плагины" → "Добавить новый"
3. Нажмите кнопку "Загрузить плагин"
4. Выберите скачанный ZIP-архив
5. Нажмите "Установить плагин"

### 2. Активация плагина

1. После установки нажмите "Активировать плагин"
2. Плагин появится в меню WordPress как "Туристическая карта"

## Использование

### 1. Импорт геоданных

1. Перейдите в "Туристическая карта" → "Импорт и экспорт"
2. Выберите файл GeoJSON для импорта
3. Нажмите "Импортировать"
4. Все существующие объекты будут удалены перед импортом

### 2. Создание объектов карты

1. Перейдите в "Туристическая карта" → "Все объекты"
2. Нажмите "Добавить новый"
3. Заполните информацию:
   - Название
   - Описание
   - Координаты (или кликните на карте)
   - Контактные данные
   - Категория
4. Сохраните объект

### 3. Отображение карты на сайте

Используйте шорткод `[kolchugino_map]` для отображения карты на страницах и записях.

## Особенности

- **Оффлайн режим**: Поддержка локальных тайлов карты
- **Интерактивная карта**: OpenLayers для отображения
- **Метки объектов**: Различные категории с цветовыми маркерами
- **Поиск и фильтрация**: Поиск по категориям
- **Экспорт данных**: Экспорт в GeoJSON и форматы
- **Мультиязычность**: Поддержка русского языка

## Требования

- WordPress 5.0 или выше
- PHP 7.4 или выше
- JavaScript включен
- MySQL 5.6 или выше

## Технические детали

- **Платформа**: WordPress
- **Картографическая библиотека**: OpenLayers 10.3.1
- **Формат данных**: GeoJSON
- **Тип поста**: `map_object`
- **Таксономия**: `map_category`

## Логирование ошибок

При включенном WP_DEBUG ошибки записываются в файл error.log:
```
[Kolchugino Map] Plugin version: 1.0.45
[Kolchugino Map Debug] AJAX HANDLERS INITIALIZED
[Kolchugino Map Export] EXPORTING GEOJSON
```
