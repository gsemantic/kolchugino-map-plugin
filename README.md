# Kolchugino Map Plugin

Туристическая карта Кольчугино — плагин для WordPress, который добавляет интерактивную карту достопримечательностей города Кольчугино.

## Описание

Плагин отображает туристические объекты на карте с использованием GeoJSON данных. Поддерживает различные типы объектов (POI) и предоставляет удобный интерфейс для навигации по городу.

## Требования

- WordPress 5.0 или выше
- PHP 7.4 или выше

## Установка

1. Скачайте архив с плагином
2. Распакуйте файлы в директорию `/wp-content/plugins/kolchugino-map-plugin/`
3. Активируйте плагин в административной панели WordPress в разделе "Плагины"
4. Добавьте шорткод `[kolchugino_map]` на нужную страницу или используйте функцию `<?php kolchugino_map_render(); ?>` в шаблоне

## Использование

### Шорткод

```
[kolchugino_map]
```

### PHP функция

```php
<?php kolchugino_map_render(); ?>
```

## Структура плагина

```
kolchugino-map-plugin/
├── assets/              # Статические ресурсы (CSS, JS, изображения)
├── data/                # GeoJSON данные с координатами объектов
├── includes/            # Основные классы плагина
├── templates/           # Шаблоны для отображения карты
├── kolchugino-map.php   # Главный файл плагина
├── README.md            # Документация
└── LICENSE              # Лицензия
```

## Данные

Файлы в директории `data/` содержат координаты и информацию о туристических объектах в формате GeoJSON:
- `kolchugino_poi.geojson` — основные точки интереса
